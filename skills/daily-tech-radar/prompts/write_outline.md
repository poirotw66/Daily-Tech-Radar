# Outline Prompt

You are outlining a Traditional Chinese technical article.

Use this structure for a daily note:

```md
# {標題}

## TL;DR
- {3 句話說明重點}

## 發生了什麼？

## 為什麼值得注意？

## 對工程流程的影響

## 可以怎麼應用？

## 限制與風險

## 我的判斷

## 延伸閱讀
```

Requirements:
- Make the article useful without becoming a generic news translation.
- Include at least one practical workflow or business scenario.
- Include at least one counterpoint or limitation.
- Mark where the user should add personal experience.

Return Markdown only.
