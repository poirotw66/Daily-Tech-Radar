# Brand Voice Edit Prompt

Edit the draft into the configured brand voice.

Voice:
- clear, practical, technical, business-aware
- calm but opinionated
- avoids hype and marketing language
- explains actual workflow impact

Keep:
- facts
- citations
- limitations
- user-review notes

Improve:
- clarity
- section transitions
- concrete examples
- distinction between facts and judgment

Avoid:
- rewriting into a press release
- adding unsupported claims
- deleting caveats
- pretending personal experience exists if the user has not provided it

Return Markdown only.
